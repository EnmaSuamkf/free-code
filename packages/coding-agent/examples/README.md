# Pi Coding Agent Examples

This directory contains examples for the Pi Coding Agent, demonstrating how to extend its functionality through the SDK and custom extensions.

## Extension Examples

Explore the `extensions/` directory to find a variety of examples showcasing how to build custom extensions. These examples cover:

-   **Lifecycle Events:** Intercept tool calls, implement safety gates, and modify context.
-   **Custom Tools:** Create new tools for managing to-do lists, asking questions, creating sub-agents, and more.
-   **UI Customization:** Add custom footers, headers, editors, and overlays.
-   **Integrations:** Connect with external tools like Git, SSH, and file watchers.
-   **And more:** Modify the system prompt, customize compaction, and even add new providers.

For more details, see the [extensions/](extensions/) directory.

## SDK Examples

The `sdk/` directory illustrates how to use Pi programmatically. These examples show how to:

-   Create and manage agent sessions.
-   Customize models, prompts, and tools.
-   Integrate extensions into your own applications.

For more details, see the [sdk/](sdk/) directory.
