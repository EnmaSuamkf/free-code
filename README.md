# free-code

Complete AI coding agent with a native macOS app, VS Code/Cursor plugin, and CLI.

---

## Quick start

Clone the repository, then open the local docs in your browser — no build step required.

```bash
git clone https://github.com/badlogic/free-code.git
```

**Open the docs in Finder:**

1. Open **Finder** and go to the cloned `free-code` folder.
2. Open the **`web-docs`** folder.
3. Double-click **`index.html`** — it opens in your default browser.

From there you can browse capabilities, commands, installation steps for macOS / VS Code / Cursor / CLI, and interactive demos. Everything you need to install and use free-code is in that site.
